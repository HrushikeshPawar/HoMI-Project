Kaṭapayādi System

It is an ancient Indian numerical encoding system where devanāgari letters are assigned numbers. Here, the consonants k, ṭ, p, y represent number 1, and kh, ṭh, ph, r represent number 2, and so on. When many letters can represent a single number, it provides the flexibility in forming meaningful words to depict numbers so that they could fit in the chosen prosody. For more details, go to Katapayadi System.

Our application will handle the decoding and encoding of this system. While decoding is done by syllable parsing and figuring out the associated number, for encoding, we have created a huge database of Sanskrit words sourced from various dictionaries. When an input number is given, matching words are retrieved from the database. When the input number doesn’t have an exact match, then it is split variously, and matching words are then retrieved from the database for the segments of the number.

Examples for the decoder:
  • गोपीभाग्यमधुव्रात-शृङ्गिशोदधिसन्धिग॥ खलजीवितखाताव गलहालारसंधर॥ - Devnagari
  • āyurārogyasaukhyam - IAST
  • jYAnaM paramaM Dyeyam - SLP1
  • ಗೀರ್ನಃ ಶ್ರೇಯಃ, ಧೇನವಃ ಶ್ರೀಃ, ರುದ್ರಸ್ತು ನಮ್ಯಃ - Kannada
