Kaṭapayādi System

{{"""It is an ancient Indian numerical system where devanāgari letters are assigned numbers. Here, letters *k, ṭ, p, y* represent number *1*, and *kh*, *ṭh*, *ph*, *r* represent number *2*, and so on. When many letters can represent a single number, it provides the flexibility in forming meaningful words to depict numbers so that they could fit in the prosody. For more details, go to [Katapayadi System - Wikipedia](https://en.wikipedia.org/wiki/Katapayadi_system)."""|markdown}}

Out application will handle the decoding and encoding of this system. For encoding, we have created a huge database of Sanskrit words. When an input number is given, matching words are retrieved from the database. When the number doesn’t have an exact match, then it is split variously, and matching words are then retrieved from the database for the segments of the numbers.
